# 회원 탈퇴 강의평가 보존 익명화 구현 계획

## 목표

강의평가를 보존하면서 회원 탈퇴를 완료하고, 재가입 계정과 탈퇴 전 평가를 분리한다.

## 작업

- [x] 강의평가와 태그가 있는 회원 탈퇴 통합 회귀 테스트를 추가한다.
- [x] Refresh Token 사용자 단위 삭제 테스트를 추가한다.
- [x] User와 Student 익명화 도메인 동작을 구현한다.
- [x] 탈퇴 서비스에서 사용자 삭제 대신 익명화와 인증 정보 정리를 수행한다.
- [x] 탈퇴 User의 기존 access token을 JWT 필터에서 차단한다.
- [x] 재가입 시 새 User와 새 Student가 생성되는지 검증한다.
- [x] 관련 테스트와 전체 테스트를 Java 17로 실행한다.

## 제외 범위

- 강의평가 조회 API의 작성자 표시 변경.
- 과거 익명 평가를 재가입 계정으로 복원하거나 병합하는 기능.
- DB 스키마와 공개 API 계약 변경.
