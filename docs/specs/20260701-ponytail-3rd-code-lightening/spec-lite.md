# Ponytail 3차 잔여 코드 경량화

## 배경

#290, #291 병합 후 최신 `dev` 기준으로 다시 점검한 결과, 동작보다 문서 보조 계층이나 미사용 helper에 가까운 코드가 남아 있다.

## 목표

- 공개 API 응답 형식, DB 스키마, 인증 동작은 바꾸지 않는다.
- 실제 참조가 없는 메서드와 중복 getter를 제거한다.
- 선택지가 local 하나뿐인 설정 스위치를 줄인다.
- OpenAPI 문서 계층은 `/v3/api-docs` 계약 테스트가 유지되는 범위에서만 축소한다.

## 검증

- 삭제 전 `rg`로 참조 여부를 확인한다.
- 관련 단위 테스트를 먼저 실행한다.
- 최종적으로 `./gradlew test`를 실행한다.
